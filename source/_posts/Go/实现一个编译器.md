解析

第一步，词法分析： 

输入，分割成一些被称为 Token 的东西，这个过程是在词法分析器（Tokenizer或者Lexer）中完成的

比如 输入 `(add 2 (subtract 4 2))`

它产生的 Token 看起来或许是这样的：

```json
[
 { type: 'paren',  value: '('        },
 { type: 'name',   value: 'add'      },
 { type: 'number', value: '2'        },
 { type: 'paren',  value: '('        },
 { type: 'name',   value: 'subtract' },
 { type: 'number', value: '4'        },
 { type: 'number', value: '2'        },
 { type: 'paren',  value: ')'        },
 { type: 'paren',  value: ')'        }
]
```



语法分析：Token转AST。
